Welcome to my blog!

April 2026

I've been meaning to start a blog for a while, mostly as a place to write things down; ideas I'm working through, notes from papers I've read, things I've learned that I don't want to forget.

I'm not sure what the cadence will look like. Probably sporadic. But I'd rather have somewhere to put things than not.

The main categories I expect to write in:

  • ML notes: reading notes, derivations, things I find interesting
  • Research thoughts: half-formed ideas and observations
  • Misc: everything else

If you're reading this, hope something here is useful or interesting to you :)
